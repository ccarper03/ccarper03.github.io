About Me

Charles is a Unity developer with proven work expirience who works with Game Studio Agencies to develope games. He has 6 years of software development experience, using Agile methodology, version control(Git/Svn/Perforce), and project managment. He also has strong development skills, well versed in Computer Science, OOP, and Game Programming Patterns. He has worked with an indie game studio "Luna Wolf Studios" on/off since 2015 and worked on 2 published games as a Unity Game Developer and a triple A gaming studio "Epic Games" as Quality Assurance on 5 released games. He is certified in Unity: Pragmatic UI, 3D Game Programming and 2D Pathfinding, as well as holds a B.S. degree in Game Design and Development.
